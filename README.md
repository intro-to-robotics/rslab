# rslab

Quarto-based website starter adapted from `course-website`.

This repository is intentionally cleaned so you can add your own content.

## Project structure

- `index.qmd`: homepage
- `notes/`: notes pages
- `slides/`: RevealJS slide decks
- `code/`: code and assets
- `styles/`: SCSS theme files

## Quick start

1. Install [Quarto](https://quarto.org/docs/get-started/).
2. Preview locally:

   ```bash
   quarto preview
   ```

3. Build static output:

   ```bash
   quarto render
   ```

Rendered output is generated in `docs/`.
